# Pilot Outreach Playbook

## Target Pilot Profiles
- Performance coaches with 10-100 active clients
- Wellness-focused clinics
- SME teams with health/wellbeing initiatives

## Offer Structure (Zero-Cost Pilot)
- Duration: 4 weeks
- Seats: 5-20 users
- Success Criteria:
  - Weekly adherence score trend
  - HRV/sleep/readiness directional improvement
  - User-reported value score (1-10)

## Outreach Message Template
Subject: 4-week precision health pilot for your team

Hi {{name}},

I built AEVUM, a health intelligence app that converts biometrics into daily, safety-aware protocols with outcome evidence reports.

I am opening a limited 4-week pilot for {{segment}} teams:
- contraindication-aware protocols
- adherence risk prediction
- weekly outcome deltas and export-ready evidence packs

If useful, I can run a 15-minute walkthrough and set up a no-cost pilot for your group.

Best,
{{your_name}}

## Weekly Pilot Cadence
- Week 0: onboarding + baseline capture
- Week 1: adherence and protocol consistency check
- Week 2: safety and intervention adjustment review
- Week 3: outcome trend review
- Week 4: final evidence report + decision call

## Data Collection Rules
- Capture minimum 5 biometric days per week per user
- Track completion status of daily protocol actions
- Run weekly export and store in pilot folder
