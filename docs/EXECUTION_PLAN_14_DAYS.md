# AEVUM 14-Day Execution Plan

## Goal
Land 3 pilot accounts, run a measurable 28-day proof cycle, and prepare an investor/enterprise-ready evidence package.

## Day 1-2: Pilot Pipeline Build
1. Build a target list of 30 leads:
- 10 performance coaches
- 10 wellness clinics
- 10 SMB employer wellness programs
2. Use `docs/PILOT_OUTREACH_PLAYBOOK.md` message template.
3. Send 15 outbound messages/day.

## Day 3-4: Demo + Qualification
1. Run 15-minute demos with qualified leads.
2. Ask and log:
- seat count,
- success KPI priority,
- risk/compliance concerns,
- start date.
3. Move 3 best-fit leads to pilot kickoff.

## Day 5-6: Pilot Onboarding
1. Follow `docs/PILOT_ONBOARDING_RUNBOOK.md`.
2. Ensure each pilot has:
- baseline biometrics,
- plan entitlement set,
- contraindication profile set,
- weekly check-in owner.

## Day 7: Baseline Evidence Snapshot
1. Run:
```powershell
cd C:\Users\Austin\OneDrive\aevum

```
2. Save output from `docs/reports/`.
3. Log pilot health in `docs/PILOT_TRACKER_TEMPLATE.csv`.

## Day 8-10: Engagement + Risk Control
1. Trigger mid-cycle review for low adherence users.
2. Push clinician note + signoff workflow for flagged users.
3. Apply contraindication profile updates where needed.

## Day 11-12: Compliance + Claims Hardening
1. Finalize artifacts in `docs/COMPLIANCE_EVIDENCE_INDEX.md`.
2. Use `docs/CLINICAL_CLAIMS_BOUNDARY_MATRIX.md` for all external messaging.
3. Export audit bundle for each pilot workspace.

## Day 13: Commercial Packaging
1. Build pilot case summaries:
- activation,
- adherence,
- risk/readiness movement,
- retention,
- ROI direction.
2. Map pricing recommendation per pilot.

## Day 14: Founder Review + Next Commitments
1. Update deck using `docs/FOUNDER_SALES_DECK_TEMPLATE.md`.
2. Prepare next 30-day roadmap:
- pilot expansion,
- clinical validation milestones,
- regulatory artifact closure.

## Success Criteria
- 3 pilots active.
- 70%+ activation in each pilot.
- 60%+ adherence completion trend.
- Measurable positive trend in readiness/risk signals.
- Evidence packet ready for buyer or investor diligence.
